# TIC TAC TOC

로그인이나 데이터베이스 없이 공유 링크 하나로 친구와 즐기는 틱택토 웹앱입니다.

## 게임 모드

- 클래식: 기본 3×3 틱택토
- 3D 큐브: 3×3×3 공간의 49개 승리선
- 초제한: 각 플레이어의 돌이 최대 3개만 유지되는 모드
- 스피드: 매 턴 8초 제한
- 혼자 연습: 컴퓨터 상대 오프라인 플레이

## 구조와 비용

브라우저 간 연결은 Trystero WebRTC P2P를 사용합니다. 게임 서버와 DB가 필요하지 않아 정적 호스팅만으로 운영할 수 있습니다. Cloudflare Pages, Netlify, Vercel 정적 배포를 사용할 수 있습니다.

## 실행

```bash
npm start
```

표시된 로컬 주소로 접속합니다. 두 브라우저에서 초대 링크를 열면 온라인 대전을 확인할 수 있습니다.

## 테스트

```bash
npm test
```
